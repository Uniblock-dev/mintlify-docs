Avalanche C-Chain RPC

Use Uniblock's JSON-RPC endpoint to access Avalanche C-Chain with low-latency routing and a single API surface across providers.

## What is Avalanche C-Chain?
Avalanche C-Chain is an EVM-compatible mainnet.

## What is the Avalanche C-Chain JSON-RPC API?
The JSON-RPC API is the standard interface for reading chain state, sending transactions, and interacting with smart contracts where supported.

## Making API requests
Send a POST request to `https://api.uniblock.dev/uni/v1/json-rpc?chainId=43114`. Include the JSON-RPC method and params in the request body.

## Supported networks
| Network | Type | Chain ID |
| --- | --- | --- |
| Avalanche C-Chain | Mainnet | `43114` |
| Avalanche Fuji Testnet | Testnet | `43113` |
