BNB Smart Chain Testnet RPC

Use Uniblock's JSON-RPC endpoint to access BNB Smart Chain Testnet with low-latency routing and a single API surface across providers.

## What is BNB Smart Chain Testnet?
BNB Smart Chain Testnet is an EVM-compatible testnet. It is a testnet for BNB Smart Chain.

## What is the BNB Smart Chain Testnet JSON-RPC API?
The JSON-RPC API is the standard interface for reading chain state, sending transactions, and interacting with smart contracts where supported.

## Making API requests
Send a POST request to `https://api.uniblock.dev/uni/v1/json-rpc?chainId=97`. Include the JSON-RPC method and params in the request body.

## Supported networks
| Network | Type | Chain ID |
| --- | --- | --- |
| BNB Smart Chain | Mainnet | `56` |
| BNB Smart Chain Testnet | Testnet | `97` |
