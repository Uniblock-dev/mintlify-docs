Kava RPC

Use Uniblock's JSON-RPC endpoint to access Kava with low-latency routing and a single API surface across providers.

## What is Kava?
Kava is an EVM-compatible mainnet.

## What is the Kava JSON-RPC API?
The JSON-RPC API is the standard interface for reading chain state, sending transactions, and interacting with smart contracts where supported.

## Making API requests
Send a POST request to `https://api.uniblock.dev/uni/v1/json-rpc?chainId=2222`. Include the JSON-RPC method and params in the request body.

## Supported networks
| Network | Type | Chain ID |
| --- | --- | --- |
| Kava | Mainnet | `2222` |
