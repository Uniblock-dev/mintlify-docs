Ink Sepolia RPC

Use Uniblock's JSON-RPC endpoint to access Ink Sepolia with low-latency routing and a single API surface across providers.

## What is Ink Sepolia?
Ink Sepolia is an EVM-compatible testnet. It is a testnet for Ink.

## What is the Ink Sepolia JSON-RPC API?
The JSON-RPC API is the standard interface for reading chain state, sending transactions, and interacting with smart contracts where supported.

## Making API requests
Send a POST request to `https://api.uniblock.dev/uni/v1/json-rpc?chainId=763373`. Include the JSON-RPC method and params in the request body.
