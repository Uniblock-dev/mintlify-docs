Zilliqa Testnet RPC

Use Uniblock's JSON-RPC endpoint to access Zilliqa Testnet with low-latency routing and a single API surface across providers.

## What is Zilliqa Testnet?
Zilliqa Testnet is an EVM-compatible testnet. It is a testnet for Zilliqa.

## What is the Zilliqa Testnet JSON-RPC API?
The JSON-RPC API is the standard interface for reading chain state, sending transactions, and interacting with smart contracts where supported.

## Making API requests
Send a POST request to `https://api.uniblock.dev/uni/v1/json-rpc?chainId=33101`. Include the JSON-RPC method and params in the request body.

## Supported networks

| Zilliqa | Mainnet | `32769` |
| Zilliqa Testnet | Testnet | `33101` |
