Soneium RPC

Use Uniblock's JSON-RPC endpoint to access Soneium with low-latency routing and a single API surface across providers.

## What is Soneium?
Soneium is an EVM-compatible mainnet.

## What is the Soneium JSON-RPC API?
The JSON-RPC API is the standard interface for reading chain state, sending transactions, and interacting with smart contracts where supported.

## Making API requests
Send a POST request to `https://api.uniblock.dev/uni/v1/json-rpc?chainId=1868`. Include the JSON-RPC method and params in the request body.
