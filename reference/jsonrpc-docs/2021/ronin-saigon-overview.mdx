Ronin Saigon RPC

Use Uniblock's JSON-RPC endpoint to access Ronin Saigon with low-latency routing and a single API surface across providers.

## What is Ronin Saigon?
Ronin Saigon is an EVM-compatible testnet. It is a testnet for RONIN.

## What is the Ronin Saigon JSON-RPC API?
The JSON-RPC API is the standard interface for reading chain state, sending transactions, and interacting with smart contracts where supported.

## Making API requests
Send a POST request to `https://api.uniblock.dev/uni/v1/json-rpc?chainId=2021`. Include the JSON-RPC method and params in the request body.

## Supported networks
| Network | Type | Chain ID |
| --- | --- | --- |
| RONIN | Mainnet | `2020` |
| Ronin Saigon | Testnet | `2021` |
