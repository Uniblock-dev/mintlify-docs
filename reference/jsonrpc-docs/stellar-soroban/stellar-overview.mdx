Stellar RPC

Use Uniblock's JSON-RPC endpoint to access Stellar with low-latency routing and a single API surface across providers.

## What is Stellar?
Stellar is a non-EVM mainnet.

## What is the Stellar JSON-RPC API?
The JSON-RPC API is the standard interface for reading chain state, sending transactions, and interacting with smart contracts where supported.

## Making API requests
Send a POST request to `https://api.uniblock.dev/uni/v1/json-rpc?chainId=stellar-soroban`. Include the JSON-RPC method and params in the request body.

## Supported networks
| Network | Type | Chain ID |
| --- | --- | --- |
| Stellar | Mainnet | `stellar-soroban` |
| Stellar Testnet | Testnet | `stellar-soroban-testnet` |
