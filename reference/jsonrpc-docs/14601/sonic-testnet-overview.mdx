Sonic Testnet RPC

Use Uniblock's JSON-RPC endpoint to access Sonic Testnet with low-latency routing and a single API surface across providers.

## What is Sonic Testnet?
Sonic Testnet is an EVM-compatible testnet. It is a testnet for Sonic Mainnet.

## What is the Sonic Testnet JSON-RPC API?
The JSON-RPC API is the standard interface for reading chain state, sending transactions, and interacting with smart contracts where supported.

## Making API requests
Send a POST request to `https://api.uniblock.dev/uni/v1/json-rpc?chainId=14601`. Include the JSON-RPC method and params in the request body.

## Supported networks
| Network | Type | Chain ID |
| --- | --- | --- |
| Sonic Mainnet | Mainnet | `146` |
| Sonic Blaze Testnet | Testnet | `57054` |
| Sonic Testnet | Testnet | `14601` |
