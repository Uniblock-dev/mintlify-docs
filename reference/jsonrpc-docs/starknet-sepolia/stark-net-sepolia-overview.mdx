StarkNet Sepolia RPC

Use Uniblock's JSON-RPC endpoint to access StarkNet Sepolia with low-latency routing and a single API surface across providers.

## What is StarkNet Sepolia?
StarkNet Sepolia is a non-EVM testnet. It is a testnet for StarkNet.

## What is the StarkNet Sepolia JSON-RPC API?
The JSON-RPC API is the standard interface for reading chain state, sending transactions, and interacting with smart contracts where supported.

## Making API requests
Send a POST request to `https://api.uniblock.dev/uni/v1/json-rpc?chainId=starknet-sepolia`. Include the JSON-RPC method and params in the request body.

## Supported networks
| Network | Type | Chain ID |
| --- | --- | --- |
| StarkNet | Mainnet | `starknet` |
| StarkNet Sepolia | Testnet | `starknet-sepolia` |
