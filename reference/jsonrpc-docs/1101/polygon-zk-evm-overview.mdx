Polygon zkEVM RPC

Use Uniblock's JSON-RPC endpoint to access Polygon zkEVM with low-latency routing and a single API surface across providers.

## What is Polygon zkEVM?
Polygon zkEVM is an EVM-compatible mainnet.

## What is the Polygon zkEVM JSON-RPC API?
The JSON-RPC API is the standard interface for reading chain state, sending transactions, and interacting with smart contracts where supported.

## Making API requests
Send a POST request to `https://api.uniblock.dev/uni/v1/json-rpc?chainId=1101`. Include the JSON-RPC method and params in the request body.

## Supported networks
| Network | Type | Chain ID |
| --- | --- | --- |
| Polygon zkEVM | Mainnet | `1101` |
| Polygon zkEVM Cardona Testnet | Testnet | `2442` |
