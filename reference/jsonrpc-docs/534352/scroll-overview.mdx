Scroll RPC

Use Uniblock's JSON-RPC endpoint to access Scroll with low-latency routing and a single API surface across providers.

## What is Scroll?
Scroll is an EVM-compatible mainnet.

## What is the Scroll JSON-RPC API?
The JSON-RPC API is the standard interface for reading chain state, sending transactions, and interacting with smart contracts where supported.

## Making API requests
Send a POST request to `https://api.uniblock.dev/uni/v1/json-rpc?chainId=534352`. Include the JSON-RPC method and params in the request body.

## Supported networks
| Network | Type | Chain ID |
| --- | --- | --- |
| Scroll | Mainnet | `534352` |
| Scroll Sepolia | Testnet | `534351` |
