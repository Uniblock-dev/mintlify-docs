Bitcoin Testnet 4 RPC

Use Uniblock's JSON-RPC endpoint to access Bitcoin Testnet 4 with low-latency routing and a single API surface across providers.

## What is Bitcoin Testnet 4?
Bitcoin Testnet 4 is a non-EVM testnet. It is a testnet for Bitcoin.

## What is the Bitcoin Testnet 4 JSON-RPC API?
The JSON-RPC API is the standard interface for reading chain state, sending transactions, and interacting with smart contracts where supported.

## Making API requests
Send a POST request to `https://api.uniblock.dev/uni/v1/json-rpc?chainId=bitcoin-testnet4`. Include the JSON-RPC method and params in the request body.

## Supported networks
| Network | Type | Chain ID |
| --- | --- | --- |
| Bitcoin | Mainnet | `bitcoin` |
| Bitcoin Testnet | Testnet | `bitcoin-testnet` |
| Bitcoin Testnet 4 | Testnet | `bitcoin-testnet4` |
