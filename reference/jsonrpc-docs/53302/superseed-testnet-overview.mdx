Superseed Testnet RPC

Use Uniblock's JSON-RPC endpoint to access Superseed Testnet with low-latency routing and a single API surface across providers.

## What is Superseed Testnet?
Superseed Testnet is an EVM-compatible testnet. It is a testnet for Superseed.

## What is the Superseed Testnet JSON-RPC API?
The JSON-RPC API is the standard interface for reading chain state, sending transactions, and interacting with smart contracts where supported.

## Making API requests
Send a POST request to `https://api.uniblock.dev/uni/v1/json-rpc?chainId=53302`. Include the JSON-RPC method and params in the request body.

## Supported networks
| Network | Type | Chain ID |
| --- | --- | --- |
| Superseed | Mainnet | `5330` |
| Superseed Testnet | Testnet | `53302` |
