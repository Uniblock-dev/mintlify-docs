Morph Holesky Testnet RPC

Use Uniblock's JSON-RPC endpoint to access Morph Holesky Testnet with low-latency routing and a single API surface across providers.

## What is Morph Holesky Testnet?
Morph Holesky Testnet is an EVM-compatible testnet. It is a testnet for Morph.

## What is the Morph Holesky Testnet JSON-RPC API?
The JSON-RPC API is the standard interface for reading chain state, sending transactions, and interacting with smart contracts where supported.

## Making API requests
Send a POST request to `https://api.uniblock.dev/uni/v1/json-rpc?chainId=2810`. Include the JSON-RPC method and params in the request body.

## Supported networks
| Network | Type | Chain ID |
| --- | --- | --- |
| Morph | Mainnet | `2818` |
| Morph Holesky Testnet | Testnet | `2810` |
