Sei Network Mainnet RPC

Use Uniblock's JSON-RPC endpoint to access Sei Network Mainnet with low-latency routing and a single API surface across providers.

## What is Sei Network Mainnet?
Sei Network Mainnet is an EVM-compatible mainnet.

## What is the Sei Network Mainnet JSON-RPC API?
The JSON-RPC API is the standard interface for reading chain state, sending transactions, and interacting with smart contracts where supported.

## Making API requests
Send a POST request to `https://api.uniblock.dev/uni/v1/json-rpc?chainId=1329`. Include the JSON-RPC method and params in the request body.

## Supported networks
| Network | Type | Chain ID |
| --- | --- | --- |
| Sei Network Mainnet | Mainnet | `1329` |
| Sei Testnet | Testnet | `1328` |
