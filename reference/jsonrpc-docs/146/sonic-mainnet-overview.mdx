Sonic Mainnet RPC

Use Uniblock's JSON-RPC endpoint to access Sonic Mainnet with low-latency routing and a single API surface across providers.

## What is Sonic Mainnet?
Sonic Mainnet is an EVM-compatible mainnet.

## What is the Sonic Mainnet JSON-RPC API?
The JSON-RPC API is the standard interface for reading chain state, sending transactions, and interacting with smart contracts where supported.

## Making API requests
Send a POST request to `https://api.uniblock.dev/uni/v1/json-rpc?chainId=146`. Include the JSON-RPC method and params in the request body.

## Supported networks
| Network | Type | Chain ID |
| --- | --- | --- |
| Sonic Mainnet | Mainnet | `146` |
| Sonic Blaze Testnet | Testnet | `57054` |
| Sonic Testnet | Testnet | `14601` |
