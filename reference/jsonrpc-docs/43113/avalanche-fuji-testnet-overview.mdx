Avalanche Fuji Testnet RPC

Use Uniblock's JSON-RPC endpoint to access Avalanche Fuji Testnet with low-latency routing and a single API surface across providers.

## What is Avalanche Fuji Testnet?
Avalanche Fuji Testnet is an EVM-compatible testnet. It is a testnet for Avalanche C-Chain.

## What is the Avalanche Fuji Testnet JSON-RPC API?
The JSON-RPC API is the standard interface for reading chain state, sending transactions, and interacting with smart contracts where supported.

## Making API requests
Send a POST request to `https://api.uniblock.dev/uni/v1/json-rpc?chainId=43113`. Include the JSON-RPC method and params in the request body.

## Supported networks
| Network | Type | Chain ID |
| --- | --- | --- |
| Avalanche C-Chain | Mainnet | `43114` |
| Avalanche Fuji Testnet | Testnet | `43113` |
