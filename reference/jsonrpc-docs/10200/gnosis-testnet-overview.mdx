Gnosis Testnet RPC

Use Uniblock's JSON-RPC endpoint to access Gnosis Testnet with low-latency routing and a single API surface across providers.

## What is Gnosis Testnet?
Gnosis Testnet is an EVM-compatible testnet. It is a testnet for Gnosis.

## What is the Gnosis Testnet JSON-RPC API?
The JSON-RPC API is the standard interface for reading chain state, sending transactions, and interacting with smart contracts where supported.

## Making API requests
Send a POST request to `https://api.uniblock.dev/uni/v1/json-rpc?chainId=10200`. Include the JSON-RPC method and params in the request body.

## Supported networks
| Network | Type | Chain ID |
| --- | --- | --- |
| Gnosis | Mainnet | `100` |
| Gnosis Testnet | Testnet | `10200` |
