Oasis Testnet RPC

Use Uniblock's JSON-RPC endpoint to access Oasis Testnet with low-latency routing and a single API surface across providers.

## What is Oasis Testnet?
Oasis Testnet is an EVM-compatible testnet. It is a testnet for Oasis.

## What is the Oasis Testnet JSON-RPC API?
The JSON-RPC API is the standard interface for reading chain state, sending transactions, and interacting with smart contracts where supported.

## Making API requests
Send a POST request to `https://api.uniblock.dev/uni/v1/json-rpc?chainId=42261`. Include the JSON-RPC method and params in the request body.

## Supported networks
| Network | Type | Chain ID |
| --- | --- | --- |
| Oasis | Mainnet | `42262` |
| Oasis Testnet | Testnet | `42261` |
