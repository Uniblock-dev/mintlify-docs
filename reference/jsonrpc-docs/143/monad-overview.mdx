Monad RPC

Use Uniblock's JSON-RPC endpoint to access Monad with low-latency routing and a single API surface across providers.

## What is Monad?
Monad is an EVM-compatible mainnet.

## What is the Monad JSON-RPC API?
The JSON-RPC API is the standard interface for reading chain state, sending transactions, and interacting with smart contracts where supported.

## Making API requests
Send a POST request to `https://api.uniblock.dev/uni/v1/json-rpc?chainId=143`. Include the JSON-RPC method and params in the request body.

## Supported networks
| Network | Type | Chain ID |
| --- | --- | --- |
| Monad | Mainnet | `143` |
| Monad Testnet | Testnet | `10143` |
