ApeChain RPC

Use Uniblock's JSON-RPC endpoint to access ApeChain with low-latency routing and a single API surface across providers.

## What is ApeChain?
ApeChain is an EVM-compatible mainnet.

## What is the ApeChain JSON-RPC API?
The JSON-RPC API is the standard interface for reading chain state, sending transactions, and interacting with smart contracts where supported.

## Making API requests
Send a POST request to `https://api.uniblock.dev/uni/v1/json-rpc?chainId=33139`. Include the JSON-RPC method and params in the request body.
