Sui RPC

Use Uniblock's JSON-RPC endpoint to access Sui with low-latency routing and a single API surface across providers.

## What is Sui?
Sui is a non-EVM mainnet.

## What is the Sui JSON-RPC API?
The JSON-RPC API is the standard interface for reading chain state, sending transactions, and interacting with smart contracts where supported.

## Making API requests
Send a POST request to `https://api.uniblock.dev/uni/v1/json-rpc?chainId=sui`. Include the JSON-RPC method and params in the request body.
