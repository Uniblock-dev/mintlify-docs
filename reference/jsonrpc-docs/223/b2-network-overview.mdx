B2 Network RPC

Use Uniblock's JSON-RPC endpoint to access B2 Network with low-latency routing and a single API surface across providers.

## What is B2 Network?
B2 Network is an EVM-compatible mainnet.

## What is the B2 Network JSON-RPC API?
The JSON-RPC API is the standard interface for reading chain state, sending transactions, and interacting with smart contracts where supported.

## Making API requests
Send a POST request to `https://api.uniblock.dev/uni/v1/json-rpc?chainId=223`. Include the JSON-RPC method and params in the request body.

## Supported networks
| Network | Type | Chain ID |
| --- | --- | --- |
| B2 Network | Mainnet | `223` |
