World RPC

Use Uniblock's JSON-RPC endpoint to access World with low-latency routing and a single API surface across providers.

## What is World?
World is an EVM-compatible mainnet.

## What is the World JSON-RPC API?
The JSON-RPC API is the standard interface for reading chain state, sending transactions, and interacting with smart contracts where supported.

## Making API requests
Send a POST request to `https://api.uniblock.dev/uni/v1/json-rpc?chainId=480`. Include the JSON-RPC method and params in the request body.

## Supported networks
| Network | Type | Chain ID |
| --- | --- | --- |
| World | Mainnet | `480` |
| World Sepolia | Testnet | `4801` |
