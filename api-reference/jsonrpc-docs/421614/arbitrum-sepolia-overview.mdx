Arbitrum Sepolia RPC

Use Uniblock's JSON-RPC endpoint to access Arbitrum Sepolia with low-latency routing and a single API surface across providers.

## What is Arbitrum Sepolia?
Arbitrum Sepolia is an EVM-compatible testnet. It is a testnet for Arbitrum.

## What is the Arbitrum Sepolia JSON-RPC API?
The JSON-RPC API is the standard interface for reading chain state, sending transactions, and interacting with smart contracts where supported.

## Making API requests
Send a POST request to `https://api.uniblock.dev/uni/v1/json-rpc?chainId=421614`. Include the JSON-RPC method and params in the request body.

## Supported networks
| Network | Type | Chain ID |
| --- | --- | --- |
| Arbitrum | Mainnet | `42161` |
| Arbitrum Sepolia | Testnet | `421614` |
