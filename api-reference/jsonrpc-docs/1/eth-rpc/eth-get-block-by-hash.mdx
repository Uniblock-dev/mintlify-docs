### Parameters

Parameters for eth_getBlockByHash: [blockHash, fullTransactions].

- `params[0]` (string): 32 Bytes - hash of the block where this transaction was included. Null when pending.
- `params[1]` (boolean): If true, returns the full transaction objects. If false, returns only the transaction hashes.

### Returns

<ResponseField body='result' type='object'>
  A block object returned by Ethereum RPC. Null if no block is found.
</ResponseField>

curl --request POST \

  --header 'content-type: application/json' \
