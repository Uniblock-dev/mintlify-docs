### Parameters

Parameters for eth_getBlockByNumber: [blockNumber, fullTransactions].

- `params[0]` (oneOf): Block identifier. Can be an integer block number or a string tag: "latest", "earliest", or "pending".
- `params[1]` (boolean): If true, returns the full transaction objects. If false, returns only the transaction hashes.

### Returns

<ResponseField body='result' type='object'>
  A block object returned by Ethereum RPC. Null if no block is found.
</ResponseField>

<RequestExample>

```bash Curl
curl --request POST \
  --url https://api.uniblock.dev/uni/v1/json-rpc?chainId=1 \
  --header 'content-type: application/json' \
  --data '{
    "id": 1,
    "jsonrpc": "2.0",
    "method": "eth_getBlockByNumber",
    "params": [
      "0x1b4",
      true
    ]
  }'
```
