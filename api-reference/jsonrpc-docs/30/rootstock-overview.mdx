Rootstock RPC

Use Uniblock's JSON-RPC endpoint to access Rootstock with low-latency routing and a single API surface across providers.

## What is Rootstock?
Rootstock is an EVM-compatible mainnet.

## What is the Rootstock JSON-RPC API?
The JSON-RPC API is the standard interface for reading chain state, sending transactions, and interacting with smart contracts where supported.

## Making API requests
Send a POST request to `https://api.uniblock.dev/uni/v1/json-rpc?chainId=30`. Include the JSON-RPC method and params in the request body.

## Supported networks
| Network | Type | Chain ID |
| --- | --- | --- |
| Rootstock | Mainnet | `30` |
| Rootstock | Testnet | `31` |
