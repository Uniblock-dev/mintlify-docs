Telos EVM RPC

Use Uniblock's JSON-RPC endpoint to access Telos EVM with low-latency routing and a single API surface across providers.

## What is Telos EVM?
Telos EVM is an EVM-compatible mainnet.

## What is the Telos EVM JSON-RPC API?
The JSON-RPC API is the standard interface for reading chain state, sending transactions, and interacting with smart contracts where supported.

## Making API requests
Send a POST request to `https://api.uniblock.dev/uni/v1/json-rpc?chainId=40`. Include the JSON-RPC method and params in the request body.

## Supported networks
| Network | Type | Chain ID |
| --- | --- | --- |
| Telos EVM | Mainnet | `40` |
