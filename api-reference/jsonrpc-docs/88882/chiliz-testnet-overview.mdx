Chiliz Testnet RPC

Use Uniblock's JSON-RPC endpoint to access Chiliz Testnet with low-latency routing and a single API surface across providers.

## What is Chiliz Testnet?
Chiliz Testnet is an EVM-compatible testnet. It is a testnet for Chiliz.

## What is the Chiliz Testnet JSON-RPC API?
The JSON-RPC API is the standard interface for reading chain state, sending transactions, and interacting with smart contracts where supported.

## Making API requests
Send a POST request to `https://api.uniblock.dev/uni/v1/json-rpc?chainId=88882`. Include the JSON-RPC method and params in the request body.
