### Parameters

<ParamField body='params.startLedger' type='number'>

</ParamField>
<ParamField body='params.order' type='string'>
  Sort order by ledger sequence.
</ParamField>
<ParamField body='params.pagination' type='object'>
  Pagination controls.
</ParamField>

### Returns

<ResponseField name='result' type='object'>
  Horizon ledgers page.
</ResponseField>

<RequestExample>

```bash Curl
curl --request POST \

  --header 'content-type: application/json' \
  --data '{
    "id": 1,
    "jsonrpc": "2.0",
    "method": "getLedgers",
    "params": {

      "pagination": {
        "limit": 10
      }
    }
  }'
```
