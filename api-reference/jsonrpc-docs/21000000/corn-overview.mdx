Corn RPC

Use Uniblock's JSON-RPC endpoint to access Corn with low-latency routing and a single API surface across providers.

## What is Corn?
Corn is an EVM-compatible mainnet.

## What is the Corn JSON-RPC API?
The JSON-RPC API is the standard interface for reading chain state, sending transactions, and interacting with smart contracts where supported.

## Making API requests
Send a POST request to `https://api.uniblock.dev/uni/v1/json-rpc?chainId=21000000`. Include the JSON-RPC method and params in the request body.

## Supported networks

| --- | --- | --- |
| Corn | Mainnet | `21000000` |
