Botanix Testnet RPC

Use Uniblock's JSON-RPC endpoint to access Botanix Testnet with low-latency routing and a single API surface across providers.

## What is Botanix Testnet?
Botanix Testnet is an EVM-compatible testnet. It is a testnet for Botanix.

## What is the Botanix Testnet JSON-RPC API?
The JSON-RPC API is the standard interface for reading chain state, sending transactions, and interacting with smart contracts where supported.

## Making API requests
Send a POST request to `https://api.uniblock.dev/uni/v1/json-rpc?chainId=3636`. Include the JSON-RPC method and params in the request body.

## Supported networks
| Network | Type | Chain ID |
| --- | --- | --- |
| Botanix | Mainnet | `3637` |
| Botanix Testnet | Testnet | `3636` |
