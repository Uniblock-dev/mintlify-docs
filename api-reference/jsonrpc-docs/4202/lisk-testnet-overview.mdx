Lisk Testnet RPC

Use Uniblock's JSON-RPC endpoint to access Lisk Testnet with low-latency routing and a single API surface across providers.

## What is Lisk Testnet?
Lisk Testnet is an EVM-compatible testnet. It is a testnet for Lisk.

## What is the Lisk Testnet JSON-RPC API?
The JSON-RPC API is the standard interface for reading chain state, sending transactions, and interacting with smart contracts where supported.

## Making API requests
Send a POST request to `https://api.uniblock.dev/uni/v1/json-rpc?chainId=4202`. Include the JSON-RPC method and params in the request body.

## Supported networks
| Network | Type | Chain ID |
| --- | --- | --- |
| Lisk | Mainnet | `1135` |
| Lisk Testnet | Testnet | `4202` |
