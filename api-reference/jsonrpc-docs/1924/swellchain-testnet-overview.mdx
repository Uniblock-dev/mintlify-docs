Swellchain Testnet RPC

Use Uniblock's JSON-RPC endpoint to access Swellchain Testnet with low-latency routing and a single API surface across providers.

## What is Swellchain Testnet?
Swellchain Testnet is an EVM-compatible testnet. It is a testnet for Swellchain.

## What is the Swellchain Testnet JSON-RPC API?
The JSON-RPC API is the standard interface for reading chain state, sending transactions, and interacting with smart contracts where supported.

## Making API requests
Send a POST request to `https://api.uniblock.dev/uni/v1/json-rpc?chainId=1924`. Include the JSON-RPC method and params in the request body.

## Supported networks
| Network | Type | Chain ID |
| --- | --- | --- |
| Swellchain | Mainnet | `1923` |
| Swellchain Testnet | Testnet | `1924` |
