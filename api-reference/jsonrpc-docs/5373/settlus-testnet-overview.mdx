Settlus Testnet RPC

Use Uniblock's JSON-RPC endpoint to access Settlus Testnet with low-latency routing and a single API surface across providers.

## What is Settlus Testnet?
Settlus Testnet is an EVM-compatible testnet. It is a testnet for Settlus.

## What is the Settlus Testnet JSON-RPC API?
The JSON-RPC API is the standard interface for reading chain state, sending transactions, and interacting with smart contracts where supported.

## Making API requests
Send a POST request to `https://api.uniblock.dev/uni/v1/json-rpc?chainId=5373`. Include the JSON-RPC method and params in the request body.

## Supported networks
| Network | Type | Chain ID |
| --- | --- | --- |
| Settlus | Mainnet | `5371` |
| Settlus Testnet | Testnet | `5373` |
