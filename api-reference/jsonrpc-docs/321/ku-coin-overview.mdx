Ku Coin RPC

Use Uniblock's JSON-RPC endpoint to access Ku Coin with low-latency routing and a single API surface across providers.

## What is Ku Coin?
Ku Coin is an EVM-compatible mainnet.

## What is the Ku Coin JSON-RPC API?
The JSON-RPC API is the standard interface for reading chain state, sending transactions, and interacting with smart contracts where supported.

## Making API requests
Send a POST request to `https://api.uniblock.dev/uni/v1/json-rpc?chainId=321`. Include the JSON-RPC method and params in the request body.

## Supported networks
| Network | Type | Chain ID |
| --- | --- | --- |
| Ku Coin | Mainnet | `321` |
| Ku Coin Testnet | Testnet | `322` |
