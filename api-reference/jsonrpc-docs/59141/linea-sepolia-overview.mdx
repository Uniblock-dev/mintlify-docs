Linea Sepolia RPC

Use Uniblock's JSON-RPC endpoint to access Linea Sepolia with low-latency routing and a single API surface across providers.

## What is Linea Sepolia?
Linea Sepolia is an EVM-compatible testnet. It is a testnet for Linea.

## What is the Linea Sepolia JSON-RPC API?
The JSON-RPC API is the standard interface for reading chain state, sending transactions, and interacting with smart contracts where supported.

## Making API requests
Send a POST request to `https://api.uniblock.dev/uni/v1/json-rpc?chainId=59141`. Include the JSON-RPC method and params in the request body.

## Supported networks
| Network | Type | Chain ID |
| --- | --- | --- |
| Linea | Mainnet | `59144` |
| Linea Sepolia | Testnet | `59141` |
