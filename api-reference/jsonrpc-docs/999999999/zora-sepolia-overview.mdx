Zora Sepolia RPC

Use Uniblock's JSON-RPC endpoint to access Zora Sepolia with low-latency routing and a single API surface across providers.

## What is Zora Sepolia?
Zora Sepolia is an EVM-compatible testnet. It is a testnet for Zora.

## What is the Zora Sepolia JSON-RPC API?
The JSON-RPC API is the standard interface for reading chain state, sending transactions, and interacting with smart contracts where supported.

## Making API requests
Send a POST request to `https://api.uniblock.dev/uni/v1/json-rpc?chainId=999999999`. Include the JSON-RPC method and params in the request body.
