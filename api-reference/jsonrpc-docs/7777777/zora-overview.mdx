Zora RPC

Use Uniblock's JSON-RPC endpoint to access Zora with low-latency routing and a single API surface across providers.

## What is Zora?
Zora is an EVM-compatible mainnet.

## What is the Zora JSON-RPC API?
The JSON-RPC API is the standard interface for reading chain state, sending transactions, and interacting with smart contracts where supported.

## Making API requests
Send a POST request to `https://api.uniblock.dev/uni/v1/json-rpc?chainId=7777777`. Include the JSON-RPC method and params in the request body.

## Supported networks

| Zora | Mainnet | `7777777` |
| Zora Sepolia | Testnet | `999999999` |
