### Parameters

Parameters for retrieving a normalized Move module.

<ParamField body='params[0]' type='string' required>
  Object ID of the Move package.
</ParamField>
<ParamField body='params[1]' type='string' required>
  The name of the Move module.
</ParamField>

### Returns

<ResponseField name='result' type='object'>
  Structured representation of a Move module.
  <Expandable title='properties'>
    <ResponseField name='fileFormatVersion' type='integer'>
      The Move binary file format version.
    </ResponseField>
    <ResponseField name='address' type='string'>
      The hexadecimal address of the package.
    </ResponseField>
    <ResponseField name='name' type='string'>
      Name of the module.
    </ResponseField>
    <ResponseField name='friends' type='array'>
      List of module IDs that this module declares as friends.
      <Expandable title='items'>
        <ResponseField name='items[]' type='string'>
          SuiMoveModuleId (formatted as <address>::<module>).
        </ResponseField>
      </Expandable>
    </ResponseField>
    <ResponseField name='structs' type='object'>
      Map of struct names to normalized struct definitions.
    </ResponseField>
    <ResponseField name='exposedFunctions' type='object'>
      Map of function names to normalized function definitions.
    </ResponseField>
  </Expandable>
</ResponseField>
