### Parameters

Create an unsigned transaction to send all SUI coins to a recipient.

<ParamField body='params[0]' type='string' required>
  The Sui address of the transaction signer.
</ParamField>
<ParamField body='params[1]' type='string[]' required>
  List of SUI coin object IDs to be used, including the gas coin.
</ParamField>
<ParamField body='params[2]' type='string' required>
  The recipient's Sui address.
</ParamField>
<ParamField body='params[3]' type='string' required>
  The gas budget for the transaction.
</ParamField>
