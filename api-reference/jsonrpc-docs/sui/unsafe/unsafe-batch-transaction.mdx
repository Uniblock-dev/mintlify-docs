### Parameters

Create an unsigned batched transaction.

<ParamField body='params[0]' type='string' required>
  The transaction signer's Sui address.
</ParamField>
<ParamField body='params[1]' type='object[]' required>
  List of transaction request parameters.
</ParamField>
<ParamField body='params[2]' type='string' required>
  Optional gas object ID to be used. If not provided, one will be chosen by the node.
</ParamField>
<ParamField body='params[3]' type='string' required>
  Gas budget (maximum gas units that can be used).
</ParamField>
<ParamField body='params[4]' type='string'>
  Transaction builder mode. Optional.
</ParamField>

### Returns

<ResponseField name='result' type='object'>
  Unsigned batched transaction block.
  <Expandable title='properties'>
    <ResponseField name='gas' type='array'>
      Gas objects used in this transaction.
    </ResponseField>
    <ResponseField name='inputObjects' type='array'>
      List of objects used as input in the transaction.
    </ResponseField>
    <ResponseField name='txBytes' type='string'>
      BCS serialized transaction data bytes (base64).
    </ResponseField>
  </Expandable>
</ResponseField>
