### Parameters

Create an unsigned transaction to stake SUI to a validator.

<ParamField body='params[0]' type='string' required>
  The Sui address of the transaction signer.
</ParamField>
<ParamField body='params[1]' type='string[]' required>
  List of Coin<SUI> object IDs to be staked.
</ParamField>
<ParamField body='params[2]' type='string' required>
  Amount to stake. If omitted, the full balance of all coins will be used.
</ParamField>
<ParamField body='params[3]' type='string' required>
  The validator's Sui address.
</ParamField>
<ParamField body='params[4]' type='string' required>
  Optional gas object ID. If not provided, a gas object will be auto-selected.
</ParamField>
<ParamField body='params[5]' type='string' required>
  Gas budget for the transaction.
</ParamField>
