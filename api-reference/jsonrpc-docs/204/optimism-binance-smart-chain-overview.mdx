Optimism Binance Smart Chain RPC

Use Uniblock's JSON-RPC endpoint to access Optimism Binance Smart Chain with low-latency routing and a single API surface across providers.

## What is Optimism Binance Smart Chain?
Optimism Binance Smart Chain is an EVM-compatible mainnet.

## What is the Optimism Binance Smart Chain JSON-RPC API?
The JSON-RPC API is the standard interface for reading chain state, sending transactions, and interacting with smart contracts where supported.

## Making API requests
Send a POST request to `https://api.uniblock.dev/uni/v1/json-rpc?chainId=204`. Include the JSON-RPC method and params in the request body.

## Supported networks
| Network | Type | Chain ID |
| --- | --- | --- |
| Optimism Binance Smart Chain | Mainnet | `204` |
| Optimism Binance Smart Chain Testnet | Testnet | `5611` |
