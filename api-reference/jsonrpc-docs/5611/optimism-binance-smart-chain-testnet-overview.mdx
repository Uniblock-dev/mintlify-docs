Optimism Binance Smart Chain Testnet RPC

Use Uniblock's JSON-RPC endpoint to access Optimism Binance Smart Chain Testnet with low-latency routing and a single API surface across providers.

## What is Optimism Binance Smart Chain Testnet?
Optimism Binance Smart Chain Testnet is an EVM-compatible testnet. It is a testnet for Optimism Binance Smart Chain.

## What is the Optimism Binance Smart Chain Testnet JSON-RPC API?
The JSON-RPC API is the standard interface for reading chain state, sending transactions, and interacting with smart contracts where supported.

## Making API requests
Send a POST request to `https://api.uniblock.dev/uni/v1/json-rpc?chainId=5611`. Include the JSON-RPC method and params in the request body.

## Supported networks
| Network | Type | Chain ID |
| --- | --- | --- |
| Optimism Binance Smart Chain | Mainnet | `204` |
| Optimism Binance Smart Chain Testnet | Testnet | `5611` |
