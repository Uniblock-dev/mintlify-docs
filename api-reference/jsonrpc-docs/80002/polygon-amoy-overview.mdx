Polygon Amoy RPC

Use Uniblock's JSON-RPC endpoint to access Polygon Amoy with low-latency routing and a single API surface across providers.

## What is Polygon Amoy?
Polygon Amoy is an EVM-compatible testnet. It is a testnet for Polygon.

## What is the Polygon Amoy JSON-RPC API?
The JSON-RPC API is the standard interface for reading chain state, sending transactions, and interacting with smart contracts where supported.

## Making API requests
Send a POST request to `https://api.uniblock.dev/uni/v1/json-rpc?chainId=80002`. Include the JSON-RPC method and params in the request body.

## Supported networks
| Network | Type | Chain ID |
| --- | --- | --- |
| Polygon | Mainnet | `137` |
| Polygon Amoy | Testnet | `80002` |
