### Parameters

Parameters: [address, block parameter].

<ParamField body='params[0]' type='string' required>
  The address.
</ParamField>
<ParamField body='params[1]' type='object'>
  Block identifier. Can be an integer block number or a string tag: "latest", "earliest", or "pending".
</ParamField>

### Returns

<ResponseField name='result' type='object'>
  Account metadata at the specified block.
  <Expandable title='properties'>
    <ResponseField name='balance' type='string'>
      Account balance in peb.
    </ResponseField>
    <ResponseField name='nonce' type='string'>
      Transaction count for the account.
    </ResponseField>
    <ResponseField name='storageRoot' type='string'>
      Storage trie root hash.
    </ResponseField>
    <ResponseField name='codeHash' type='string'>
      Hash of the account code.
    </ResponseField>
  </Expandable>
</ResponseField>
