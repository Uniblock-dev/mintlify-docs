### Parameters

Parameters: [blockHash, fullTransactions].

<ParamField body='params[0]' type='string' required>
  32 Bytes - hash of the block where this transaction was included. Null when pending.
</ParamField>
<ParamField body='params[1]' type='boolean' required>
  If true, includes full transaction objects instead of hashes.
</ParamField>

### Returns

<ResponseField name='result' type='object'>
  A Klaytn block object. Null if the block cannot be found.
</ResponseField>
