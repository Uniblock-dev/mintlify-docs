### Parameters

<Expandable title='Default parameters'>
  <ParamField query='chainId' type='string' required default='5000'>
    Unique identifier for a blockchain network.
  </ParamField>
  <ParamField body='id' type='number' required default={1}>
    Request identifier used to match responses.
  </ParamField>
  <ParamField body='jsonrpc' type='string' required default='2.0'>
    JSON-RPC version.
  </ParamField>
  <ParamField body='method' type='string' required default='eth_estimateGas'>
    JSON-RPC method name.
  </ParamField>
</Expandable>

<ParamField body='params' type='object' required placeholder='[{},0]'>
  Two-element array: [modified transaction object, default block]. The transaction omits "nonce" and allows optional "from".
  
  Positional parameters for <code>eth_estimateGas</code>:
  <ul>
  <li><code>params[0]</code>: Legacy transaction (pre-EIP-2718).</li>
  <li><code>params[1]</code>: Block identifier. Can be an integer block number or a string tag: "latest", "earliest", or "pending".</li>
  </ul>
</ParamField>

### Returns

<ResponseField name='result' type='string'>

</ResponseField>
