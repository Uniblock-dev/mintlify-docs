### Parameters

Two-element array: [modified transaction object, default block]. The transaction omits "nonce" and allows optional "from".

<ParamField body='params[0]' type='object' required>
  Legacy transaction (pre-EIP-2718).
</ParamField>
<ParamField body='params[1]' type='object' required>
  Block identifier. Can be an integer block number or a string tag: "latest", "earliest", or "pending".
</ParamField>

### Returns

<ResponseField name='result' type='string'>
  The amount of gas used. Example: <code>"0x5208"</code>.
</ResponseField>

```bash Curl
curl --request POST \
  --url https://api.uniblock.dev/uni/v1/json-rpc?chainId=36900 \
  --header 'content-type: application/json' \
  --data '{
    "id": 1,
    "jsonrpc": "2.0",
    "method": "eth_estimateGas",
    "params": [
      {},
      0
    ]
  }'
```
