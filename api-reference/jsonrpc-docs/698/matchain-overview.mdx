Matchain RPC

Use Uniblock's JSON-RPC endpoint to access Matchain with low-latency routing and a single API surface across providers.

## What is Matchain?
Matchain is an EVM-compatible mainnet.

## What is the Matchain JSON-RPC API?
The JSON-RPC API is the standard interface for reading chain state, sending transactions, and interacting with smart contracts where supported.

## Making API requests
Send a POST request to `https://api.uniblock.dev/uni/v1/json-rpc?chainId=698`. Include the JSON-RPC method and params in the request body.

## Supported networks

| Matchain | Mainnet | `698` |
