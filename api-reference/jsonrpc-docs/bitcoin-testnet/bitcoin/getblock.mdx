### Parameters

Parameters: [blockHash, verbosity?].

<ParamField body='params[0]' type='string' required>
  32 Bytes - hash of the transaction.
</ParamField>
<ParamField body='params[1]' type='number'>
  Verbosity level (0-3) controlling JSON detail.
</ParamField>

### Returns

<ResponseField name='result' type='object'>
  Block data or raw hex depending on verbosity.
</ResponseField>
