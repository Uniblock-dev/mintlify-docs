### Parameters

<Expandable title='Default parameters'>
  <ParamField query='chainId' type='string' required default='397'>
    Unique identifier for a blockchain network.
  </ParamField>
  <ParamField body='id' type='number' required default={1}>
    Request identifier used to match responses.
  </ParamField>
  <ParamField body='jsonrpc' type='string' required default='2.0'>
    JSON-RPC version.
  </ParamField>
  <ParamField body='method' type='string' required default='tx'>
    JSON-RPC method name.
  </ParamField>
</Expandable>

<ParamField body='params' type='object' required placeholder='["0xD70952032620CC4E2737EB8AC379806359D8E0B17B0488F627997A0B043ABDED",true]'>
  The parameters for method tx: [transactionHash, prove].
  
  Positional parameters for <code>tx</code>:
  <ul>
  <li><code>params[0]</code>: 32 Bytes - Hash of the transaction.</li>
  <li><code>params[1]</code>: To include proofs into the response.</li>
  </ul>
</ParamField>

### Returns

<ResponseField name='result' type='object'>
  The transactions with or without proof information.
</ResponseField>
