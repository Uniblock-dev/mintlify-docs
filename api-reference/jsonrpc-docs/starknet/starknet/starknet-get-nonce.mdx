### Parameters

Parameters for starknet_getNonce: [block_id, contract_address].

<ParamField body='params.block_id' type='object' required>
  BlockID object containing hash and number.
</ParamField>
<ParamField body='params.contract_address' type='string' required>
  The address of the request contract.
</ParamField>

### Returns

<ResponseField name='result' type='string'>
  The last nonce used for the requested contract. Example: <code>"0x1c"</code>.
</ResponseField>

```bash Curl
curl --request POST \

  --header 'content-type: application/json' \

    "id": 1,
    "jsonrpc": "2.0",

      },
      "contract_address": "0x0555ec3ba8aea37915c4cf850b4f9d33e4a0a1caa0ad3953d725860e38fa6e17"
    }
  }'
```
