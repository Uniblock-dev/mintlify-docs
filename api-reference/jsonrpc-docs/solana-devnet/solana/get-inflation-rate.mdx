### Parameters

<Expandable title='Default parameters'>
  <ParamField query='chainId' type='string' required default='solana-devnet'>
    Unique identifier for a blockchain network.
  </ParamField>
  <ParamField body='id' type='number' required default={1}>
    Request identifier used to match responses.
  </ParamField>
  <ParamField body='jsonrpc' type='string' required default='2.0'>
    JSON-RPC version.
  </ParamField>
  <ParamField body='method' type='string' required default='getInflationRate'>
    JSON-RPC method name.
  </ParamField>
</Expandable>

This method does not require any parameters.

### Returns

<ResponseField name='result' type='object'>
  Breakdown of inflation allocation for a specific epoch.
  <Expandable title='properties'>
    <ResponseField name='total' type='number'>
      Total inflation rate (as a decimal).
    </ResponseField>
    <ResponseField name='validator' type='number'>
      Portion of inflation allocated to validators.
    </ResponseField>
    <ResponseField name='foundation' type='number'>
      Portion of inflation allocated to the foundation.
    </ResponseField>
    <ResponseField name='epoch' type='integer'>
      Epoch for which the inflation values apply.
    </ResponseField>
  </Expandable>
</ResponseField>

```bash Curl
curl --request POST \
  --url https://api.uniblock.dev/uni/v1/json-rpc?chainId=solana-devnet \
  --header 'content-type: application/json' \
  --data '{
    "id": 1,
    "jsonrpc": "2.0",
    "method": "getInflationRate",
    "params": []
  }'
```

</RequestExample>
