### Parameters

  <ParamField query='chainId' type='string' required default='solana-devnet'>
    Unique identifier for a blockchain network.
  </ParamField>
  <ParamField body='id' type='number' required default={1}>
    Request identifier used to match responses.
  </ParamField>
  <ParamField body='jsonrpc' type='string' required default='2.0'>
    JSON-RPC version.
  </ParamField>
  <ParamField body='method' type='string' required default='getVersion'>
    JSON-RPC method name.
  </ParamField>

This method does not require any parameters.

### Returns

<ResponseField name='result' type='object'>
  Solana software version information.
  <Expandable title='properties'>
    <ResponseField name='solana-core' type='string'>
      Software version of solana-core.
    </ResponseField>
    <ResponseField name='feature-set' type='integer'>
      Unique identifier of the current software feature set.
    </ResponseField>
  </Expandable>
</ResponseField>

```bash Curl
curl --request POST \

  --header 'content-type: application/json' \
  --data '{
    "id": 1,
    "jsonrpc": "2.0",
    "method": "getVersion",
    "params": []
  }'
```
