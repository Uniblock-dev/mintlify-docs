Anime Sepolia RPC

Use Uniblock's JSON-RPC endpoint to access Anime Sepolia with low-latency routing and a single API surface across providers.

## What is Anime Sepolia?
Anime Sepolia is an EVM-compatible testnet. It is a testnet for Anime.

## What is the Anime Sepolia JSON-RPC API?
The JSON-RPC API is the standard interface for reading chain state, sending transactions, and interacting with smart contracts where supported.

## Making API requests
Send a POST request to `https://api.uniblock.dev/uni/v1/json-rpc?chainId=6900`. Include the JSON-RPC method and params in the request body.

## Supported networks
| Network | Type | Chain ID |
| --- | --- | --- |
| Anime | Mainnet | `69000` |
| Anime Sepolia | Testnet | `6900` |
