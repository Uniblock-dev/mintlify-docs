Plasma Testnet RPC

Use Uniblock's JSON-RPC endpoint to access Plasma Testnet with low-latency routing and a single API surface across providers.

## What is Plasma Testnet?
Plasma Testnet is an EVM-compatible testnet. It is a testnet for Plasma.

## What is the Plasma Testnet JSON-RPC API?
The JSON-RPC API is the standard interface for reading chain state, sending transactions, and interacting with smart contracts where supported.

## Making API requests
Send a POST request to `https://api.uniblock.dev/uni/v1/json-rpc?chainId=9746`. Include the JSON-RPC method and params in the request body.

## Supported networks
| Network | Type | Chain ID |
| --- | --- | --- |
| Plasma | Mainnet | `9745` |
| Plasma Testnet | Testnet | `9746` |
