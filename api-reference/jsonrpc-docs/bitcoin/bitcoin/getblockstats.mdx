### Parameters

Parameters: [hash_or_height, metrics?].

<ParamField body='params[0]' type='string' required>
  Block hash or height.
</ParamField>
<ParamField body='params[1]' type='string[]'>
  Optional list of metrics to compute.
</ParamField>

### Returns

<ResponseField name='result' type='object'>
  Selected block statistics (size, weight, fees, etc.).
</ResponseField>
