Bitcoin RPC

Use Uniblock's JSON-RPC endpoint to access Bitcoin with low-latency routing and a single API surface across providers.

## What is Bitcoin?
Bitcoin is a non-EVM mainnet.

## What is the Bitcoin JSON-RPC API?
The JSON-RPC API is the standard interface for reading chain state, sending transactions, and interacting with smart contracts where supported.

## Making API requests
Send a POST request to `https://api.uniblock.dev/uni/v1/json-rpc?chainId=bitcoin`. Include the JSON-RPC method and params in the request body.

## Supported networks
| Network | Type | Chain ID |
| --- | --- | --- |
| Bitcoin | Mainnet | `bitcoin` |
| Bitcoin Testnet | Testnet | `bitcoin-testnet` |
| Bitcoin Testnet 4 | Testnet | `bitcoin-testnet4` |
