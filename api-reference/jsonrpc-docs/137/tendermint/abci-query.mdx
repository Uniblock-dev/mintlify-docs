### Parameters

Query parameters for method abci_query in the order: [path, data, height, prove]

<ParamField body='params[0]' type='string' required>
  Path to the data (e.g., "/a/b/c")
</ParamField>
<ParamField body='params[1]' type='string' required>
  Data content to query
</ParamField>
<ParamField body='params[2]' type='number' required default={0}>
  Block height. 0 means latest.
</ParamField>
<ParamField body='params[3]' type='boolean' required default={false}>
  Whether to include proofs of transaction inclusion.
</ParamField>

### Returns

<ResponseField name='result' type='object'>
  Application query response.
</ResponseField>
