### Parameters

Transaction search parameters in the order: [query, prove, page, per_page, order_by, match_events]

<ParamField body='params[0]' type='string' required>
  A transaction query string (e.g., "tx.height=1000").
</ParamField>
<ParamField body='params[1]' type='boolean' default={false}>
  Include proofs of the transaction inclusion in the block.
</ParamField>
<ParamField body='params[2]' type='number' default={1}>
  Page number (1-based).
</ParamField>
<ParamField body='params[3]' type='number' default={30}>
  Number of entries per page (max: 100).
</ParamField>
<ParamField body='params[4]' type='string' default='asc'>
  Order of transaction sorting ("asc" or "desc"), by height & index.
</ParamField>
<ParamField body='params[5]' type='boolean' default={false}>
  Match attributes in query within events in addition to height & txhash.
</ParamField>

### Returns

<ResponseField name='result' type='object'>
  The transactions returned with the search queries.
</ResponseField>

<RequestExample>

```bash Curl
curl --request POST \
  --url https://api.uniblock.dev/uni/v1/json-rpc?chainId=137 \
  --header 'content-type: application/json' \
  --data '{
    "id": 1,
    "jsonrpc": "2.0",
    "method": "tx_search",
    "params": [
      "tx.height=1000",
      true,
      1,
      30,
      "asc",
      true
    ]
  }'
```

</RequestExample>
