---
title: 'eth_getRootHash'
api: 'POST https://api.uniblock.dev/uni/v1/json-rpc'
playground: 'interactive'
---
### Parameters

Parameters: [fromBlock, toBlock]

<ParamField body='params[0]' type='number' required>
  The starting block number (inclusive).
</ParamField>
<ParamField body='params[1]' type='number' required>
  The ending block number (inclusive).
</ParamField>

### Returns

<ResponseField name='result' type='string'>
  Hex-encoded root hash covering the specified block range. Example: <code>"0x67518d9fc6bf9ff886ba1a4e932a0d0a8a1318b3f300518773aaf2210410cf73"</code>.
</ResponseField>

<RequestExample>

```bash Curl
curl --request POST \
  --url https://api.uniblock.dev/uni/v1/json-rpc?chainId=137 \
  --header 'content-type: application/json' \
  --data '{
    "id": 1,
    "jsonrpc": "2.0",
    "method": "eth_getRootHash",
    "params": [
      1000,
      1032
    ]
  }'
```

</RequestExample>
<ResponseExample>

```json Success
{
  "jsonrpc": "2.0",
  "id": 1,
  "result": "0x67518d9fc6bf9ff886ba1a4e932a0d0a8a1318b3f300518773aaf2210410cf73"
}
```

</ResponseExample>
