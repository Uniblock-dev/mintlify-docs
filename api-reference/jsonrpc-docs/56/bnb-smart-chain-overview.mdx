BNB Smart Chain RPC

Use Uniblock's JSON-RPC endpoint to access BNB Smart Chain with low-latency routing and a single API surface across providers.

## What is BNB Smart Chain?
BNB Smart Chain is an EVM-compatible mainnet.

## What is the BNB Smart Chain JSON-RPC API?
The JSON-RPC API is the standard interface for reading chain state, sending transactions, and interacting with smart contracts where supported.

## Making API requests
Send a POST request to `https://api.uniblock.dev/uni/v1/json-rpc?chainId=56`. Include the JSON-RPC method and params in the request body.

## Supported networks
| Network | Type | Chain ID |
| --- | --- | --- |
| BNB Smart Chain | Mainnet | `56` |
| BNB Smart Chain Testnet | Testnet | `97` |
