Rise Testnet RPC

Use Uniblock's JSON-RPC endpoint to access Rise Testnet with low-latency routing and a single API surface across providers.

## What is Rise Testnet?
Rise Testnet is an EVM-compatible testnet. It is a testnet for Rise.

## What is the Rise Testnet JSON-RPC API?
The JSON-RPC API is the standard interface for reading chain state, sending transactions, and interacting with smart contracts where supported.

## Making API requests
Send a POST request to `https://api.uniblock.dev/uni/v1/json-rpc?chainId=11155931`. Include the JSON-RPC method and params in the request body.
