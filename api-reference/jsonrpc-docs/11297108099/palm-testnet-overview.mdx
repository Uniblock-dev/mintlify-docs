Palm Testnet RPC

Use Uniblock's JSON-RPC endpoint to access Palm Testnet with low-latency routing and a single API surface across providers.

## What is Palm Testnet?
Palm Testnet is an EVM-compatible testnet. It is a testnet for Palm.

## What is the Palm Testnet JSON-RPC API?
The JSON-RPC API is the standard interface for reading chain state, sending transactions, and interacting with smart contracts where supported.

## Making API requests
Send a POST request to `https://api.uniblock.dev/uni/v1/json-rpc?chainId=11297108099`. Include the JSON-RPC method and params in the request body.
